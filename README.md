## Usage

To read more about classes and other references used by `animated_transform`, see the [API Reference](https://pub.dev/documentation/animated_transform/latest/).

```dart
AnimatedTransform(
  scale: 1.5,
  rotate: 45,
  flipX: true,
  flipY: false,
  child: const FlutterLogo(size: 100),
)
```
